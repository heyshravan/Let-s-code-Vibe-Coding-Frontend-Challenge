## 🌟 Key Features

### 🗺️ Interactive DSA & Tech Roadmaps
* Tracks for **C++**, **Java**, **Python**, and **Full-Stack Web Development**.
* Real-time interactive progress indicators with custom checkboxes.

### 📝 AI Resume Studio
* Interactive **ATS Score Calculator** (0–100) that updates instantly on inputs.
* Live compliance checklists highlighting grammar, formatting, and structural issues.
* Direct download buttons for premium, ATS-ready resume templates.

### 🏢 Authentic Company Prep Center
* Tailored interview preparation cards for 12 industry leaders.
* Official corporate SVG brand assets mapped cleanly to each company.
* Smart **auto-inverting CSS styling** to convert dark-colored SVG logos to solid white in dark mode.
* Fast, lightweight offline fallback to initials when offline.

### 💬 Live AI Mock Interviews
* Chat simulator to practice company-specific interview prompts.
* Interactive text checking indicating correct or incorrect responses instantly.

### 👥 Communities & Job Matcher
* Direct links to discord groups of like-minded developers.
* **Job Matcher**: Upload your PDF/DOCX resume to instantly query active job boards.

## 💻 Getting Started

### Prerequisites
* **Node.js** 18.x or higher
* **npm** or **yarn**

### Local Development

1. **Install required packages**:
   ```bash
   npm install
   ```

2. **Spin up the development server**:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the application.

## 🐳 Docker Deployment

The application features full multi-stage production container configuration.

### Build and Run using Docker CLI

1. **Build the production container**:
   ```bash
   docker build -t lets-code-app .
   ```

2. **Run the container**:
   ```bash
   docker run -p 3000:3000 lets-code-app
   ```

### Running with Docker Compose

To build and run in background with a single command:
```bash
docker compose up -d
```
